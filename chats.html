Cara Install Plugin

1. Download plugin dari Discord: discord.gg/HuGtbRvD

2. Simpan ke: C:\Users\[Nama]\AppData\Local\Roblox\Plugins\

3. Roblox Studio → Manage Plugin → Enable HTTP Requests + Script Injection

4. Klik tombol NEXUS AI di toolbar → CONNECT

5. Status hijau = terhubung! Chat langsung membangun di place Anda.
